# Release Notes: v2.1.0

`v2.1.0` is a minor release focused on devtools UI polish and stable install pinning.

## Highlights

- Improved the devtools topbar so narrow viewports no longer create horizontal page overflow.
- Tightened mobile sidebar overlay behavior with dynamic viewport height and safer backdrop offsets.
- Updated the dashboard quick-action copy to match the current workspace-oriented UI.
- Bumped package metadata from `2.0.0` to `2.1.0`.
- Updated README and MkDocs release navigation so MCP host examples can pin `@v2.1.0`.

## Upgrade Notes

Pin MCP host installs to the new release tag:

```json
{
  "command": "uvx",
  "args": [
    "--python",
    "3.11",
    "--from",
    "git+https://github.com/khoavu882/graphbase-memories-mcp@v2.1.0",
    "graphbase",
    "serve"
  ]
}
```

No graph schema migration is required for this release.
